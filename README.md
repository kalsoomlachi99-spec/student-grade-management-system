# Student Grade Management System (C++)

A C++ project that calculates student grades, percentages, and remarks based on subject marks. This repository contains three versions of the project, demonstrating my progress in C++ programming from basic implementation to a more structured and feature-rich solution.

## Features

- Calculate total marks
- Calculate percentage
- Assign grades automatically
- Generate remarks based on grades
- Input validation
- Support for multiple students
- Store student records using structures
- Sort students by percentage
- Display Top 3 Position Holders
- Display complete student records
- Formatted output using `iomanip`
